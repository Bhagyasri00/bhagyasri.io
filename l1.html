Python for Data Analysis

Here, you'll find projects showcasing how Python helps analyze data. From cleaning and organizing data to creating insightful visualizations, Python makes it easier to understand and use data for making informed decisions.

Munnar Data Analysis

This project looks at hotels in Munnar. It checks things like prices, ratings, and where hotels are. It found that most hotels are in the city center, not near Attukal Waterfalls. Hotels near the falls have prices in a smaller range. Also, better-rated hotels usually cost more. The project helps hotels understand what guests like and how to set prices.

Airbnb Data Analysis

This project looks at a dataset with about 49,000 observations about Airbnb properties in NYC. We want to understand what makes Airbnb successful in the city. We'll study things like where the properties are, what types they are, how they're priced, what guests think, and what amenities they offer. By analyzing this data, we hope to find useful information to help Airbnb keep growing and improving in NYC
